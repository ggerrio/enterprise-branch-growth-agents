# Vessel Bank — Enterprise Multi-Agent Growth Strategy Suite

Welcome to the **Vessel Bank Enterprise Multi-Agent Growth Strategy Suite**. This project implements an intelligent multi-agent network designed to help Branch Managers optimize used car finance strategy. It leverages specialized sub-agents, a local Model Context Protocol (MCP) data server, database CRUD tools, and a strict Human-in-the-Loop (HITL) governance guardrail.

---

## 🚀 Key Features

1. **Visual Database Editor (CRUD Tab):** An interactive Streamlit dashboard allowing managers to edit the branch's macro indicators and dealer sales metrics for the current month in real time, automatically recalculating totals.
2. **Live Agent Network Simulator (Simulator Tab):** Trigger a collaborative debate among 3 distinct agents:
   - **@Volume-Analyst-Agent:** Evaluates sales volume contributions and recommends loyalty program rewards or partnership reviews.
   - **@Risk-Auditor-Agent:** Audits credit risk and enforces a strict NPL threshold limit of **2.0%** to safeguard the branch.
   - **@Branch-Strategist-Orchestrator:** Correlates performance trends to staff count drops and consolidates the strategic evaluation report.
3. **HITL Governance Guardrail (Governance Tab):** Review the strategy draft and formally **Approve** or **Reject** the recommendation, automatically logging decisions to `audit_trail.json` with secure timestamps.

## 🛠️ Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ggerrio/enterprise-branch-growth-agents.git
   cd enterprise-branch-growth-agents
   ```

2. **Set up Virtual Environment:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\Activate.ps1
   ```

3. **Install Dependencies:**
   ```bash
   pip install streamlit pandas rich google-adk
   ```

---

## 💻 How to Run

### Run the Web Dashboard
```bash
.venv\Scripts\streamlit.exe run app.py
```

### Run the CLI Simulator
```bash
.venv\Scripts\python.exe simulate_committee.py
```

## 🛡️ Governance & Security Policies
* **Zero Ambient Authority:** The agents are completely blocked from executing financial budgets or making database updates autonomously.
* **HITL Gateway:** All strategies generated by the orchestrator are locked with a `[STATUS: WAITING HUMAN APPROVAL]` flag. The Branch Manager must explicitly review the report and click **Approve** or **Reject** to update the strategy status and commit the action to the audit trail log.
